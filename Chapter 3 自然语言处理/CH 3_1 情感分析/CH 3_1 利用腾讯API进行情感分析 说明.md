## 2. 准备二：用pycharm打开project

1. 把`CH 3_1 情感分析`这个文件夹下的文件，下载到本地，保存在同一个文件夹。
2. 把需要分析的数据的excel文件也保存在该文件夹。
3. 右键该文件夹，用pycharm打开，`open folder as pycharm project`


## 3. 准备三：安装需要的模块
